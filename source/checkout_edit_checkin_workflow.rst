``Checkout --> Edit --> Checkin``
=====================================

1. 在文件管理其中打开目标目录

2. 打开 ``powerShell`` 命令行快速进入目标目录

``shift + 鼠标右键`` -> 单击 ``PowerShellウインドウをここで開く（S）``

3. 键入命令 ``ess -s``

4. 键入命令 ``ess cho`` , 根据提示键入id号, 用 ``space`` 隔开

5. 回到文件浏览器双击打开文件, 进行编辑

6. 回到 ``powerShell`` 键入 ``ess chi`` 或者 ``ess chi id1 id2``
